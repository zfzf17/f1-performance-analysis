# F1 Race Performance Analysis

Exploratory data analysis on Formula 1 telemetry and timing data using the FastF1 API.

## Focus Areas
- Lap time and driver performance comparison
- Tyre compound analysis: short-run (quali pace) and long-run (race pace)
- Fuel correction and tyre degradation modelling
- Stint length simulation

## Tools & Libraries
- Python
- FastF1
- Pandas
- NumPy
- Matplotlib
- SciPy

## Data Source
FastF1 API — official F1 timing and telemetry data

## Notes
Sessions and drivers used are examples and can be changed to any race or driver of your choice.
